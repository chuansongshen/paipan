# 中国国内 AI 算命 / 算卦服务平台与价格调研

调研日期：2026-03-20  
调研范围：以中国大陆可见的公开样本为主，重点覆盖中国区 App Store、官方模型价格页、监管通报与行业报道。  
说明：部分微信小程序、公众号和私域服务的价格需要登录或先完成引流后才展示，因此本文更适合用于判断市场结构、公开价格带与商业模式，而不是做完整的全网价格普查。

## 一、核心结论

1. 这个赛道已经不是“算命软件”单一形态，而是 `独立 App + 微信生态 + 私域咨询 + 低代码智能体平台` 的混合市场。
2. 消费端收费已经形成比较清晰的价格带：
   - 低门槛体验：`5~19 元`
   - 次卡 / 季会员：`39~99 元`
   - 年会员：`88~336 元`
   - 长周期 / 永久版 / 多年版：`128~898 元`
3. 上游 AI 调用成本远低于终端售价。以一次 `2000 输入 Token + 3000 输出 Token` 的咨询为例，模型成本大多只在 `0.002~0.164 元/次` 之间。
4. 因此，这个行业的真正壁垒不在模型，而在：
   - 获客渠道
   - 报告模板与提示词包装
   - 支付转化设计
   - 私域复购
   - 合规规避能力
5. 合规风险明显偏高。中央网信办已公开通报过提供有偿算命、占卜并宣扬封建迷信的 App，这意味着“公域强分发 + 直接宣称有偿算命”的模式存在较大平台治理风险。

## 二、市场上的主要平台形态

### 1. 独立 App

这是当前最清晰、最可量化的一层。

特点：
- 便于做订阅、内购、次卡、虚拟币。
- 容易承载长报告、连续对话、历史记录、会员权益。
- 更适合做“AI 解读 + 工具箱 + 学习内容”的复合型产品。

常见产品方向：
- 八字 / 四柱 / 紫微 / 黄历工具型
- AI 塔罗 / 情绪疗愈 / 泛心理型
- 传统排盘工具升级为“AI 解读版”

### 2. 微信小程序 / 公众号 / H5

这是流量效率最高、支付最顺手的入口之一，但也是审核和内容治理风险更高的一层。

行业报道显示，这类服务长期存在于微信生态中，很多产品会先用“免费测算”“领取报告”“今日运势”引流，再引导付费解锁完整内容或转到私域。

我的判断是：
主流玩法不是直接把自己包装成“高客单价算命平台”，而是更常以“传统文化工具”“运势测算”“心理启发”“AI 解读助手”等形式出现。

这属于基于公开样本的推断，不是平台官方明示规则。

### 3. 私域咨询平台

这类平台通常不是纯 AI 服务，而是：

- 先用 AI / 算法 / 免费工具获客
- 再转成真人咨询、1 对 1 答疑、命理老师服务、起名、择吉、风水周边等更高客单业务

代表方向如 `帮帮测` 一类平台。它的核心收入不一定来自 AI 本身，而是来自后续的咨询服务、会员和虚拟币。

### 4. MaaS / 智能体平台

这是供给侧的“建站工具层”，不是最终消费入口，但对行业影响很大。

根据 2025-02-15 的公开报道，个人开发者已经可以直接通过：

- MaaS 模型服务
- `COZE`
- `文心智能体平台`

快速搭出一个 AI 应用，再接入飞书、企业微信、微信、公众号、抖音等入口。报道中受访者给出的落地成本区间是 `数百元到数万元`，上线周期可快至 `3 到 5 天`。

这意味着：
这个赛道的技术门槛已经足够低，市场更像是“包装、内容、流量、合规”的竞争，而不是“谁先掌握模型能力”的竞争。

以下价格均来自 2026-03-20 前可访问的公开页面。

| 产品 | 产品形态 | 公开价格样本 | 价格结构判断 |
|---|---|---|---|
| 元象八字-AI八字解读 | 八字 AI 解读 App | 页面说明写 `28 元/月`、`336 元/年`；App 内购还出现 `168/268 元/年`、`128 元永久会员` 等多档 SKU | 明显在做订阅实验，属于“月费 + 年费 + 永久版”并存 |

| 诸葛八字排盘 | 八字排盘 + 智能解读 App | `88 元/季度会员`、`188 元/年度会员`、`288 元/永久会员`；道具包 `7~159 元` | 典型“工具订阅 + 虚拟道具”模式 |

### 从样本看出的定价规律

1. `5~19 元` 主要用于第一次付费转化。
2. `39~99 元` 主要用于次卡、季会员、功能包。
3. `168~336 元` 是常见年费带。
4. `288 元以上` 开始进入“重度用户 / 专业用户 / 长周期承诺”的区间。
5. 高价并不一定意味着“最准”，很多时候只是：
   - 去广告
   - 解锁更多报告
   - 增加连续对话次数
   - 加历史记录、课程或学习内容
   - 升级到更完整的命盘分析

## 四、消费者端的主流收费模型

### 1. 免费排盘 + 付费解读

这是最常见的结构。

用户先免费输入出生信息或问题，拿到基础结果，再付费解锁：

- 深度报告
- 更长篇幅解释
- 合盘 / 桃花 / 财运 / 流年等专题模块
- 连续追问能力

`查八字®` 就是这个模式的典型样本：免费排盘，但付费后才能拿到长篇 AI 命运解析报告。

### 2. 单次咨询 / 次卡

这类模式更适合塔罗、问答型算卦。

优势：
- 首单门槛最低
- 容易配合社交传播
- 能快速验证用户是否愿意为“情绪价值”付费

样本中，`月神塔罗` 的 `5 元/次`、`39 元/10 次` 就很典型。

### 3. 订阅制

常见价格带：

- 月费：`19~39 元`
- 季费：`49~98 元`
- 年费：`88~336 元`

订阅制适合的产品不是“算一次就走”的工具，而是强调：

- 每日运势
- 长期陪伴
- 连续追问
- 多模块工具箱
- 学习内容

### 4. 永久版 / 多年版

这类模式常用于更专业的排盘工具。

本质不是卖一次结果，而是卖：

- 重度使用许可
- 更高级的工具能力
- 专业用户身份

从样本看，`128~288 元永久版` 和 `898 元/5 年` 都已经出现。

### 5. 会员 + 虚拟道具

这说明产品已经开始向游戏化和平台化走。

虚拟币 / 念珠 / 测币的作用通常是：

- 单次解锁功能
- 补充咨询次数
- 购买专题报告
- 拉高 ARPPU

这种模式比单纯订阅更灵活，也更适合做活动和价格测试。

## 五、上游 AI 平台与技术成本

如果从“做这门生意”的角度看，真正重要的不是用户付多少钱，而是你每次生成一份报告到底要花多少钱。

### 1. 官方模型价格样本

| 平台 | 公开价格 | 备注 |
|---|---|---|
| DeepSeek API | 输入 `2 元/百万 Token`，输出 `3 元/百万 Token`，缓存命中输入 `0.2 元/百万 Token` | 官方文档 |
| 阿里云百炼 `qwen-turbo` | 输入 `0.3 元/百万 Token`，输出 `0.6 元/百万 Token` | 官方帮助中心 |
| 阿里云百炼 `qwen-plus` | 输入 `2.936 元/百万 Token`，输出 `8.807 元/百万 Token` | 官方帮助中心 |
| 阿里云百炼 `qwen-max` | 输入 `11.743 元/百万 Token`，输出 `46.971 元/百万 Token` | 官方帮助中心 |
| 火山引擎豆包主力模型 | `0.0008 元/千 Token`，约 `0.8 元/百万 Token` | 火山引擎开发者社区文章 |

### 2. 单次咨询的粗略成本测算

假设一次问答平均消耗：

- 输入：`2000 Token`
- 输出：`3000 Token`

则单次模型成本大致为：

| 平台 / 模型 | 单次成本估算 |
|---|---|
| 阿里云百炼 `qwen-turbo` | 约 `0.0024 元` |
| 豆包主力模型 | 约 `0.004 元` |
| DeepSeek | 约 `0.013 元` |
| 阿里云百炼 `qwen-plus` | 约 `0.032 元` |
| 阿里云百炼 `qwen-max` | 约 `0.164 元` |

### 3. 这意味着什么

1. 如果产品卖的是 `5 元/次`，模型成本通常只占很小一部分。
2. 真正吃利润的不是 API，而是：
   - 投流
   - 支付通道
   - 客服与退款
   - 内容审核
   - 私域转化
   - 可能的真人老师服务
3. 从商业角度看，AI 算命 / 算卦服务的核心不是“模型贵不贵”，而是“你能不能把极低技术成本包装成用户愿意持续购买的体验”。

## 六、行业供给为什么会越来越多

公开报道已经非常明确地说明，这门生意的供给门槛在快速下降。

一方面，模型便宜。  
另一方面，低代码平台让非技术人员也能快速上线。

因此会出现三个结果：

1. 同质化会非常严重。
2. 大量产品会套用同一批模板话术、提示词和报告结构。
3. 真正能跑出来的，不是“最准”的，而是：
   - 最会做首单转化的
   - 最会做社交传播的
   - 最会做私域复购的
   - 最能规避平台治理风险的

## 七、合规与平台风险

这是这个行业最不能忽视的部分。

中央网信办在 2023-12-29 的公开通报中明确点名 `天天吉历 APP`：

- 在“运势测算”等版块提供有偿算命、占卜服务
- 宣扬封建迷信
- 被认定违反《网络安全法》《网络信息内容生态治理规定》等相关规定

这意味着至少可以得出两个判断：

1. 直接把产品定义为“有偿算命 / 占卜服务”，在主流公域平台上存在较强监管和审核风险。
2. 更容易存活的产品表述，往往会向以下方向靠拢：
   - 传统文化工具
   - 命理解读助手
   - 运势记录工具
   - 心理启发 / 情绪陪伴
   - 人生建议 / 泛心理成长

第二点是基于现有产品命名方式和监管样本做出的推断，不是官方豁免规则。

## 八、商业模式判断

综合样本后，我的判断是：

### 1. C 端不是“高技术壁垒生意”，而是“高包装生意”

用户为之买单的主要不是算法本身，而是：

- 仪式感
- 情绪价值
- 解释的可读性
- 报告的完整度
- 连续对话的陪伴感

### 2. 最稳的变现方式不是纯卖一次结果，而是做复购

更可持续的结构通常是：

- 首单低价
- 会员续费
- 专题报告加购
- 真人服务升级
- 私域长期陪伴

### 3. “AI 纯自动化”不是终点，“AI + 真人”更像现实生意

尤其在命理、塔罗、咨询这类场景里，用户经常买的不是结论，而是被倾听、被解释、被安抚。

这也是为什么很多平台最终都会向：

- 真人老师
- 1 对 1 咨询
- 连麦问答
- 高价定制报告

继续延展。

## 九、如果要入场，建议怎么定价

以下部分属于基于公开样本的推断建议，不是现成市场事实。

如果做一个中国市场的 AI 算命 / 算卦产品，我认为更合理的价格带可以是：

| 产品层 | 建议价格 | 用途 |
|---|---|---|
| 首单体验 | `4.9~9.9 元` | 降低第一次付费门槛 |
| 单次深度解读 | `19~39 元` | 做利润和体验平衡 |
| 次卡包 | `39~99 元` | 提高客单和复购 |
| 月会员 | `19~39 元` | 适合连续运势 / 陪伴式产品 |
| 年会员 | `168~298 元` | 适合工具箱 + 长期使用 |
| 真人升级服务 | `99 元以上` | 承接高意向用户 |

这样做的原因很简单：

- 和现有样本价格带一致，不会显得离谱
- 留出后续加购空间
- 技术成本足够低，毛利空间充分
- 用户更容易接受“先低价试，再升级”

## 十、最终判断

这个赛道在中国国内已经具备三个非常鲜明的特征：

1. `技术成本低`
2. `终端售价高于技术成本很多`
3. `平台合规风险高`

所以它不是一个“拼技术”的市场，而是一个：

- 拼包装
- 拼分发
- 拼话术
- 拼私域
- 拼风险控制

的市场。

如果只是从赚钱效率看，这类服务确实存在空间。  
但如果从长期稳定经营看，最大的变量不是模型，也不是算法，而是平台治理和内容边界。
