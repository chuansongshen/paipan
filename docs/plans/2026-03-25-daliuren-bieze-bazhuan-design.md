# 大六壬三传补齐别责与八专设计

**目标**

1. 修正大六壬在“无贼克、无比用、无遥克”场景下误落 `昴星法` 的问题。
2. 补齐当前缺失的 `别责课` 与 `八专课` 三传取法。
3. 以确定性回归测试覆盖本次修复，避免后续再次回归。

**问题现状**

- 当前 `src/utils/daliuren.js` 的 `getSanChuan` 在无贼克、无比用时，会直接转入遥克，再继续回落到 `昴星法`。
- 代码中没有实现 `别责课` 与 `八专课` 专门分支。
- 这会导致部分“不备课 / 两课无克”场景被误判。
- 例如 `2026-02-13 21:30` 当前排出 `戌未午`，但仓库自带古课数据明确对应：
  - `戊午日第十二局 干上午`
  - `课体：别责，不备，六仪`
  - 文案明确落点为 `寅午午`

**规则依据**

- 别责课：
  - 四课不全，且无上下克、无遥克。
  - 刚日取“干合上神”为初传。
  - 柔日取“支前三合”为初传。
  - 中末并取“干上神”。
- 八专课：
  - 四课只有两课，且无克。
  - 刚日取“日阳顺数三位”为初传。
  - 柔日取“辰阴逆数三位”为初传。
  - 中末并取“干上神”。

**实现方案**

## 判课分派

- 保持现有 `伏吟 -> 贼克/比用/涉害 -> 遥克` 主流程不变。
- 仅在“无贼克、无比用、无遥克”分支中插入：
  - 先判 `别责`
  - 再判 `八专`
  - 最后回落 `昴星`

## 辅助函数

- 增加四课完整度分析：
  - 判断四课上神唯一值数量
  - 判断是否为“三课”或“两课”
- 增加 `五合`、`三合前神`、`顺逆数三位` 等小函数。
- 保持函数短小，避免把 `getSanChuan` 再堆成一个超长分支块。

## 错误处理与日志

- 新增课式分支取传失败时抛出明确错误信息。
- `getSanChuan` 在分支回退时打印 `[DaLiuRen]` 前缀告警，便于后续定位。

**影响范围**

- 修改 `src/utils/daliuren.js` 的三传取法。
- 新增回归测试，覆盖：
  - 阳日别责
  - 阴日别责
  - 八专
  - 现有参考课例与伏吟回归

**验证**

- `2026-02-13 21:30` 应由 `戌未午` 修正为 `寅午午`。
- 其它别责/八专样例应命中对应课式。
- 现有大六壬参考课例与伏吟回归继续通过。
