## 当前发现
- `AI 算命/算卦` 的主流形态已从“纯排盘工具”演化为四类：
  1. 独立 App（八字、塔罗、紫微等）
  2. 微信小程序 / 公众号 / H5
  3. 私域咨询平台（AI/工具获客 + 真人转化）
  4. MaaS / Agent 平台快速搭建后接入微信、飞书、抖音等入口

- 新浪科技 2025-02-15 报道中，受访从业者提到：
  - 个人可通过云厂商和模型厂商的 MaaS 服务调用大模型 API，再制作前端应用发布。
  - 也可通过 `COZE`、`文心智能体平台` 等低代码平台快速构建并集成到飞书、企业微信、微信、公众号、抖音等入口。
  - 构建门槛低，成本“数百元到数万元”，快则 “3 到 5 天”。
  - 来源：https://finance.sina.com.cn/tech/roll/2025-02-15/doc-inekpsyk5841439.shtml

- 其他样本特征：
  - `查八字®` 明确采用“免费排盘 + 付费 2 万字 AI 解读报告”的产品结构，但公开页面未直接展示人民币单次解锁价格。
  - `帮帮测` 以真人咨询为主，同时用免费小工具和会员、虚拟币包做转化，属于“算法/AI + 真人服务”的混合平台。

- 上游模型成本显著低于 C 端零售价：
  - DeepSeek 官方文档：`输入 2 元/百万 Token`、`输出 3 元/百万 Token`，缓存命中输入更低至 `0.2 元/百万 Token`
    - 来源：https://api-docs.deepseek.com/zh-cn/quick_start/pricing
  - 阿里云百炼官方：
    - `qwen-turbo`：输入 `0.3 元/百万 Token`，输出 `0.6 元/百万 Token`
    - `qwen-plus`：输入 `2.936 元/百万 Token`，输出 `8.807 元/百万 Token`
    - `qwen-max`：输入 `11.743 元/百万 Token`，输出 `46.971 元/百万 Token`
    - 来源：https://help.aliyun.com/zh/model-studio/model-pricing
  - 火山引擎开发者社区文章：豆包主力模型 `0.0008 元/千 Token`，约等于 `0.8 元/百万 Token`
    - 来源：https://developer.volcengine.com/articles/7387976236955828236

- 基于 `输入 2000 Token + 输出 3000 Token` 的粗略测算：
  - `qwen-turbo` 单次约 `0.0024 元`
  - `豆包` 单次约 `0.004 元`（按总 5000 Token 估算，未区分输入输出）
  - `DeepSeek` 单次约 `0.013 元`
  - `qwen-plus` 单次约 `0.032 元`
  - `qwen-max` 单次约 `0.164 元`
  - 结论：模型调用成本通常远低于 `5~99 元` 的用户端单次/次卡收费。

- 合规风险非常高：
  - 中央网信办 2023-12-29 通报，`天天吉历 APP` 在“运势测算”等版块提供有偿算命、占卜服务并宣扬封建迷信，被认定违反相关规定。
  - 来源：https://www.cac.gov.cn/2023-12/29/c_1705519843935413.htm

- 竞争壁垒更多在以下环节，而不在模型本身：
  - 渠道获取流量的成本
  - 支付转化设计
  - 提示词和报告模板打磨
  - 私域复购与真人服务承接
  - 平台审核与合规包装
